BigBrain

A computational neuroscience toolset built in the Google Cloud

Attention: BigBrain is currently released as "Proof of Concept". Some features may be incomplete, as work is currently progressing on them.

Explore

Ever wondered what a neuroscience simulation looks like as it is running? Interact with your simulations and inspect them in real-time, all from right here in your browser.

Expand

The cloud is a big place. By leveraging technologies like Google AppEngine and Compute Engine, BigBrain allows researchers to run simulations in a massively parallel manner. Run hundreds or thousands of copies of a simulator, each with its own unique parameter set.

Collaborate

BigBrain makes it easy to share results and experiments with fellow researchers. Imagine being able to follow a link in a scientific paper and instantly be able to reproduce the results.

Parallel Parameter Search

Researchers set the parameters for a given simulation, run the model for a period of time, and analyze the results. If the results aren't what the researcher is expecting, the simulation is re-parameterized and repeated.

Conventional = Serial

Conventional computational neuroscience is usually a serial process even on a supercomputer!

Cloud Parallelism

BigBrain turns this concept on its head by allowing multiple simulations to run in parallel. Explore huge parameter spaces all at once and generate results in parallel in real time.

Open Source

BigBrain is an open source project, built on open standards like Javascript, HTML5 and WebGL. Feel free to download the source code and try it yourself. We designed BigBrain so that it can run on any cluster of computers and is not specific to any platform. Want to run it on your local machine? Go head! Google's Cloud? Sure!
